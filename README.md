# Anuj Kulkarni — Personal Portfolio

Simple personal portfolio website built with HTML, CSS, and JavaScript.

## Live Site

- Hosted on Vercel: https://anuj-kulkarni-portfolio.vercel.app/

## What This Portfolio Includes

- About me section
- Skills section
- Projects section
- Blog highlight
- Experience timeline
- Contact form

## Tech Stack

- HTML
- CSS
- JavaScript
- Web3Forms (for contact form)

## Run Locally

1. Clone the repo
2. Open `index.html` in your browser

## Main Files

- `index.html` — page structure
- `styles.css` — styling and layout
- `script.js` — interactivity and theme behavior
